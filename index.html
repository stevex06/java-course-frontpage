Welcome to the Course

Welcome to your learning journey in database programming using Oracle Database, SQL, and PL/SQL. Throughout this course, you will develop the knowledge and technical competencies needed to create, retrieve, organize, manipulate, and manage data using industry-standard database technologies.


Learning activities are designed to strengthen your analytical thinking, problem-solving abilities, and technical skills through lectures, laboratory exercises, hands-on programming activities, and real-world database applications.

Course Description

This course focuses on Structured Query Language (SQL) and Procedural Language/SQL (PL/SQL), equipping learners with the essential knowledge and practical skills required to build, implement, develop, test, and maintain database applications using Oracle technology.


Students will gain experience in designing databases, writing SQL queries, developing PL/SQL programs, and administering Oracle databases while applying industry-standard practices.

Learning Goal

This course aims to develop competent learners capable of designing, building, managing, and integrating Oracle database systems using SQL and PL/SQL.


  • Design relational databases.
  • Write SQL queries for data retrieval.
  • Manipulate database records efficiently.
  • Develop PL/SQL programs and procedures.
  • Apply database design principles.
  • Practice responsible database administration.
  • Prepare for industry certification and IT careers.
